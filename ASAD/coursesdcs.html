Course Outlines

Core Subjects

CS110 Software Applications

(Top)
This module gives students a basic understanding of how to work with Microsoft Office
in the Microsoft Windows environment with extensive hands-on practice.
The package includes MS Word; MS Excel; MS Powerpoint and MS Access.

CS111 Computers and Information Processing

(Top)
This module describes and explains:
  • The characteristics of computer hardware and their functions;
  • The social and economic implications of the various ranges of computer systems;
  • The different types of computer networks and data communication standards;
  • The types of software and computer languages used in current computer systems;
  • The various data structures, file organization and limitations of the physical media on
    which data are stored;
  • The functions of an operating system and the facilities offered to a programmer;
  • The features of common data processing systems;
  • The operations, development and control activities within a typical data processing department.

CS112 Programme Design

(Top)
The module describes and explains how to:
  • Apply principles and activities of system analysis to the programme specific stage;
  • Apply current programme design methods;
  • Understand and use different data structures in programmes;
  • Use available techniques and tools for testing;
  • Document programmes according to common data standards.

CS113 Mathematics for Computing

(Top)
This module teaches and explains how to:
  • Convert numbers to various bases and perform simple binary arithmetic operations;
  • Use algebraic terminology to solve algebraic equations;
  • Perform simple statistical calibrations;
  • Use Boolean, Venn diagram and logic networks;
  • Use appropriate methods in determining accuracy in computations.

CS114 Web Publishing with HTML, JavaScript and Applets (Project)

(Top)
 This unit provides students with practical experience in designing,
developing, testing, implementing and documenting a set of programmes using
HTML, JavaScript and Applets.

CS115 C Programming (Project)

(Top)
 This unit provides students with practical experience in designing, developing, testing, implementing and documenting a set of programmes using C programming language.

CS116 Structured Query Language

(Top)
This module describes and explains:
  • The relational aspects of Oracle RDBMS;
  • How to retrieve data from an Oracle database;
  • An overview of SQL and PL/SQL;
  • Character, number and date functions;
  • How to display data from multiple tables.

CS153 Introduction to Java

(Top)
This module describes and explains:
  • How to create a Java Programme;
  • How to use data, methods, classes and objects;
  • Input selection and repetition;
  • Arrays and Strings;
  • Applets.

CS154 Data Structures in C

(Top)
This module describes and explains:
  • The use of Array records and Union;
  • Pointers and dynamic variables;
  • The use of data structures such as Link List, Stacks, Queue and Binary Tree;
  • Searching and sorting.

CS163 Computer Networking

(Top)
This module introduces students to computer networks and data communication. It describes and explains:
  • The fundamentals of data and signals;
  • The different media, interfacing and modems;
  • Multiplexing;
  • The fundamentals of Local Area Network (LAN);
  • The fundamentals of Wide Area Networks (WAN).

CS164 Network Security

(Top)
The module gives an overview of network security and describes and explains:
  • How to identify security threats;
  • Security plans across different platforms.
