Course Outlines

Core Subjects

CS211 Systems Analysis & Design

(Top)
This module describes and explains:
  • An in-depth understanding of the process of systems analysis;
  • System modelling;
  • Important analysis tools and techniques;
  • Systems design;
  • Data design;
  • System testing

CS 212 Computer Architecture & Data Communication

(Top)
       This module covers computer hardware, instruction format, addressing method,
    stacks/subroutine, I/O systems and interrupts, memory organization and data communication. It also
    describes and explains:
  • Areas of specialized computing and data communication;
  • Applications which require specialized hardware configuration and data communication;
  • Recent developments in data communication.

CS213 Software Engineering

(Top)
The module describes and explains how to:
  • Software Engineering paradigms;
  • Requirements analysis fundamentals;
  • Alternatives, analysis fundamentals;
  • Alternatives, analysis techniques and formal methods;
  • Software design;
  • Programming languages;
  • Data flow oriented design;
  • Data oriented design methods;
  • Software quality assurance;
  • Software testing techniques;
  • Debugging; approaches and tools

CS214 Java Application Programming

(Top)
This module teaches and explains how to:
  • Java programming from basic to intermediate level including interface design;
  • Various constructions of data structures in Java;
  • Manipulation of data structure using different programming techniques;
  • Object-oriented programming.

CS215 C Programming JavaScript and Applets (Project)

(Top)
 This module provides the foundations for C language. It also
provides an understanding of various constructions of data structures
in C programming and consists of intensive practical workshops for students
to practice writing moderately complex programmes.

CS216 Introduction to Oracle: SQL & PL/SQL

(Top)
  • Relational and object oriented database concepts;
  • SQL plus commands;
  • Loops, control structures and explicit cursors;
  • Runtime errors.

CS217 Data Modelling & Relational Database Design

(Top)
This module describes and explains:
  • The use of ER models;
  • Developing business rules and constraints;
  • Translation of ER models to database design;
  • Optimization of design for performance

CS218 Discrete Mathematics

(Top)
This module describes and explains:
  • The relationship between mathematics and computer science;
  • The core of mathematical terminology and concepts;
  • How to use cognitive skills in logical problem solving and making deductions.

CS219 Information Management

(Top)
This module describes and explains:
  • The different types of information and their use for decision making and control process;
  • The categories of information systems and the challenges facing information systems in business activities;
  • The activities involved in managing information systems;
  • The relationship between business and information systems, and how to perform enterprise analysis and identify critical success factors

CS220 System Development (Project)

(Top)
This module explains:
  • The techniques and tools used in decision making and project management;
  • How to analyse and manage a project;
  • How to apply management techniques to the business environment;
  • How to draw on individual experience of computer systems in a range of industrial, scientific, commercial, educational and operational environments.

CS254 Logic Design

(Top)
The module explains:
  • Computer logic and digital systems;
  • The principles used in all digital systems and arithmetic operation in computers;
  • Recent developments in computer architecture.

CS255 Object Oriented Programming in C++

(Top)
The module describes and explains:
  • The concepts and drawbacks of structured programming;
  • Abstract data types using C++;
  • Object oriented design concepts;
  • Advanced use of classes.

CS256 Cognitive Science

(Top)
The module describes and explains:
  • How the human reasoning process works and forms a link with artificial intelligence;
  • Fuzzy knowledge and uncertain knowledge representation;
  • Reasoning with Fuzzy Knowledge;
  • Human and computer vision.

CS260 Advanced Networking

(Top)
The module describes and explains:
  • Networking technologies-the basic concepts of data communications, networking and connectivity;
  • Network installation and configuration;
  • Telecommunication services and network products.

CS261 Networks and the Internet

(Top)
The module explains:
  • The concepts of standard technologies used today in networking for computer communications;
  • Java & Java related technologies.

CS262 Computer Security

(Top)
The module gives an overview of network security and describes and explains:
  • How to identify security threats;
  • Security plans across different platforms.
